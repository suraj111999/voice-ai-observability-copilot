# Demo Script (2–5 Minutes)

Record with Loom showing your face + screen.

## Intro (30s)

> "This is the Voice AI Observability Copilot — embedded inside HighLevel. It ingests real Voice AI transcripts from my sandbox test calls, scores them against KPIs, and surfaces actionable recommendations."

## 1. Show GHL Integration (30s)

- Open the copilot from **HighLevel left navigation** (Custom Page or Custom Menu Link)
- Mention: "This is an iframe embed inside my sandbox sub-account, not a standalone app"

## 2. Dashboard — Unified View (60s)

- Dashboard shows **your real test call contacts** (not demo names)
- Point out: **Total Calls**, **Average Score**, **Agent Performance** table
- Highlight **Issue Heatmap** — "Objection Handling and CTA are our top failures"
- Scroll to **Use Actions** — "These are specific transcript moments needing human intervention"

## 3. Agent Analysis (60s)

- Click your **real Voice AI agent → Analyze**
- Show KPI configuration (greeting, qualification, objection handling, CTA, compliance)
- Read a **Recommendation**: objection handling prompt change
- Click into a **low-scoring test call**

## 4. Call Insight (60s)

- Show transcript with red **Use Action** flags
- Walk through KPI breakdown — "Objection handling scored 0% — agent didn't acknowledge pricing concern"
- Point to suggested intervention text

## 5. Live Ingest (30s, optional)

- Place a **live test call** to your Voice AI agent while screen recording
- Show the workflow (click Sync from GHL)
- Refresh dashboard — new call appears with real transcript

## Close (15s)

> "The validation flywheel: real transcripts → score → recommend → improve script → monitor again. Built with Vue.js and Node.js, embedded in HighLevel via Custom Page."
